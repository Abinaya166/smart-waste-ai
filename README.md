
# Smart Waste Segregation and Recycling System

An intelligent system that automatically identifies and segregates waste materials into appropriate categories to improve recycling efficiency and reduce environmental pollution.


---

## 📌 Project Overview

The **Smart Waste Segregation and Recycling System** is designed to automate the process of waste classification and sorting.
The system uses sensors and image processing techniques to detect and classify waste materials such as **biodegradable and non-biodegradable waste**.

This helps improve waste management efficiency and supports sustainable environmental practices.

---

## 🚀 Features

* Automatic waste detection and classification
* Segregation of biodegradable and non-biodegradable waste
* Sensor-based hardware integration
* User-friendly web interface
* Real-time waste monitoring

## ⚙️ Installation and Setup

### Prerequisites

* Python installed
* Arduino IDE
* Node.js (optional for frontend tools)

### Steps to Run the Project

1. Clone the repository

```bash
git clone https://github.com/your-username/smart-waste-segregation-system.git
```

2. Navigate to the project folder

```bash
cd smart-waste-segregation-system
```

3. Install required dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

5. Open the browser and visit

```
http://localhost:5000
```
